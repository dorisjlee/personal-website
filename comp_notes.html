Computing Notes

Various notes for scientific python, UNIX, and other miscellaneous programming stuff.

The idea for this page started when I was working on an astronomy project as part of Google Summer of Code. After countless of hours piling through Stackoverflow and reading READMEs, I came to realize that a lot of the edgey stuff in software and programming questions are covered by case, especially when it comes to machine-dependent issues like software installation. So I am adding my tiny bit to the community, to write my personal notes on some of the questions that I couldn't find a workable solution out there in the web.

  • Installation on Factory Resetted Mac OSx Mavericks
  • Building software packages from source without administrator rights
  • Typing Traditional Chinese in LaTeX
  • Useful IPython notebook sharing and presentation features
