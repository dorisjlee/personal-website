Computing Notes

Various notes for scientific python, UNIX, and other miscellaneous programming/software stuff.

The idea for this page started when I was working my Google Summer of Code project. After countless of hours piling through Stackoverflow and reading READMEs, I came to realize that a lot of the nitty-gritty stuff in software and programming questions are covered by case-by-case, especially when it comes to machine-dependent issues like software installation. So I am plan to write about the tips and tricks that I figure out along the way or address some questions that I couldn't find a single, complete,workable solution out there in the web. I hope you find it helpful in some way or another, please send any questions, comments, errata ..etc to: dorisjunglinlee@gmail.com.
