Computing Notes

Various notes for scientific python, UNIX, and other miscellaneous programming/software stuff.

The idea for this page started when I was working my GSoC project. After countless of hours piling through Stack Overflow and various READMEs, I came to realize that a lot of the nitty-gritty stuff in software and programming questions are covered by case-by-case, especially when it comes to machine-dependent issues like software installation. So I plan to write about the tips and tricks that I figure out along the way or to address questions with no single, complete,workable solution published out there in the web. I hope you find these helpful in some way. For any questions, please email me or leave a comment using Disqus.
